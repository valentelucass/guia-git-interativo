GUIA BÁSICO DE GIT

Comandos essenciais para controle de versão

  • git init Cria um novo repositório Git
  • git clone [url] Clona um repositório remoto na sua máquina
  • git add [nome_do_arquivo] Adiciona um arquivo no stage
  • git add . Adiciona todos os arquivos no stage
  • git commit -m [mensagem] Salva alterações no histórico
  • git status Mostra o estado do diretório
  • git pull Baixa atualizações do repositório remoto
  • git push Envia alterações para o repositório remoto
  • git remote -v Lista os repositórios remotos configurados
  • git branch [nome_da_branch] Cria uma nova branch
  • git checkout [nome_da_branch] Troca para outra branch
  • git checkout -b [nome_da_branch] Cria e altera para a branch criada
  • git merge [nome_da_branch] Junta alterações da branch com a atual
  • git log --oneline Exibe o histórico de forma resumida
  • git diff Mostra a diferença entre arquivos e a stage area
  • git log Exibe o histórico de commits
  • git restore [nome_do_arquivo] Descarta as alterações em um arquivo
  • git revert [commit_id] Desfaz um commit sem reescrever o histórico
  • git reset [nome_do_arquivo] Desfaz o git add de um arquivo
